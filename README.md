# Gems_Shop 💎

Ce projet est un e-commerce de vente de pierres fines et précieuses.

Le site est réalisé en Python avec le framework Django et la base de donnée SQLite.

## ⚙️ Les fonctionnalités :

**Mot de passe oublié :**
> Reception d'un lien de validation par e-mail qui renvoie vers une formulaire de changement de passe.

**Validation de l'inscription en 2 étapes (validation après réception de l'email)**
> Après l'inscription, envoi d'un email permettant la validation de son compte

**Choix de plusieurs variations par produits :**
> - Carat
> - Couleur

**Gestion des sessions**
> Panier sauvegardé
>
> Connexion sauvegardée
